== Installation ==

= From your WordPress dashboard =

1. Visit 'Plugins > Add New'
2. Search for 'Cut down uploads size'
3. Activate the plugin from your Plugins page.

= From WordPress.org =

1. Download 'Cut down uploads size'.
2. Upload the 'Cut down uploads size' directory to your '/wp-content/plugins/' directory, using your favorite method (ftp, sftp, scp, etc...)
3. Activate 'Cut down uploads size' from your Plugins page.
